# Fuji-Fy Shared Contracts

This directory defines the cross-platform processing contract used by:

- Python engine (`pipeline.py` + `core/engine.py`)
- Desktop bridge
- React Native app layer

Files:

- `contracts/processing.ts`: canonical TypeScript interfaces.
- `contracts/processing.schema.json`: runtime JSON schema for request payloads.
- `contracts/presets.json`: preset keys and parameter values for UI/engine parity.
